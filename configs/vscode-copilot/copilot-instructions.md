# Claude-Mem Project Memory

Use Claude-Mem memory when the task depends on previous project context, decisions, bugs, commands, or user preferences. Prefer `cmem_search` first, then `cmem_timeline` or `cmem_get_observations` when a result needs more context.

At the end of meaningful work, record durable project outcomes with `cmem_remember_change` or `cmem_remember_decision`. Do not store secrets, credentials, tokens, or private personal data.
